### 如何使用

1.  **克隆仓库**: `git clone https://github.com/petemeng/Awesome-Scientific-Charts.git`
2.  **浏览项目**: 每个项目文件夹 (如 `Projects/01_...`) 都是一个独立的单元。
3.  **运行代码**: 打开项目内的 `.R` 文件，根据代码注释安装所需包，即可复现结果。
